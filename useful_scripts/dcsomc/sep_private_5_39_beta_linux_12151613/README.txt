Requirements:
1. Place the SEP package at the same folder level as the installer.

Unix
====

Usage:
Run the installer without any options for an user interactive installation.
$ ./sep-installer.sh

Run "./sep-installer.sh -h" to check out the non-interactive cli options.

Windows
=======

Additional Requirements:
1. PowerShell 4.0 or later

Usage:
Run the installer without any options for an user interactive installation.
$ sep-installer.cmd

Run "sep-installer.cmd /?" to check out the non-interactive cli options.


Refer "SEP Installation Guide.pdf" for manual installation steps.
